# sport_tube

Приложение для управления упражнениями

Из реализованного:
  - просмотр упражнения
  - добавление упражнения
  - редактирование упражнения

Поддерживается на Android и IOS

Разработка происходила с помощью пакетов
   - drift
   - rivepod
   - go_router
   - path
   - path_provider
   - build_runner

Архитектура:
   - MVVM
